2021.10.24

  1. 文章字数统计和阅读时间预计功能下线
    原因是字数统计会把渲染 Aplayer 或渲染代码也计算在内，如果有文章包含一个不使用 meting JS 的 Aplayer 播放列表，那么统计出的文字量是一个非常庞大的数字，进而影响时间预计。为了不给读者造成误导，此功能下线
  2. 全站默认 CC 协议变更为 BY-NC-SA 4.0 请知悉
  3. 修复了 Aplayer 一个小 CSS Bug
